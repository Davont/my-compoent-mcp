# xxx 设计规范总览（占位）

本目录用于沉淀与 my-design 组件库强绑定的 `xxx 设计规范`，供 Skills 做“规范驱动的页面生成/校验”。

## 建议覆盖范围

- 布局与栅格：页面结构、栅格系统、间距规则
- 排版：字号/行高/字重/标题层级
- 颜色与主题：语义色、品牌色、暗黑模式
- 组件状态：hover/active/disabled/loading/error/success
- 表单与校验：错误提示文案、校验触发时机
- 可访问性：键盘交互、ARIA、焦点可见性

## 与 token 的关系

- 规范应尽量引用 token（而不是写死数值），例如：`--md-space-2`、`--md-font-size-14`。
